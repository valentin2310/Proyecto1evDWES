Detailed Description

Handler sends logs to Telegram using Telegram Bot API.

How to use: 1) Create a Telegram bot with https://telegram.me/BotFather; 2) Create a Telegram channel or a group where logs will be recorded; 3) Add the created bot from step 1 to the created channel/group from step 2.

In order to create an instance of TelegramBotHandler use

  1. The Telegram bot API key from step 1
  2. The channel name with the @ prefix if you created a public channel (e.g. @my_public_channel), or the channel ID with the -100 prefix if you created a private channel (e.g. -1001234567890), or the group ID from step 2 (e.g. -1234567890).
