79 protected function handleTransactionException(Throwable $e, $currentAttempt, $maxAttempts)
80 {
81 // On a deadlock, MySQL rolls back the entire transaction so we can't just
82 // retry the query. We have to throw this exception all the way out and
83 // let the developer handle it in another way. We will decrement too.
