\doxysubsection{Detailed Description}
An implementation of Delimiter\+Processor\+Interface that dispatches all calls to two or more other Delimiter\+Processors depending on the length of the delimiter run. All child Delimiter\+Processors must have different minimum lengths. A given delimiter run is dispatched to the child with the largest acceptable minimum length. If no child is applicable, the one with the largest minimum length is chosen.
