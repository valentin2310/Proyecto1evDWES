ramsey/uuid is a PHP library for generating and working with universally unique identifiers (UUIDs).

This project adheres to a code of conduct. By participating in this project and its community, you are expected to uphold this code.

Much inspiration for this library came from the \href{http://docs.oracle.com/javase/6/docs/api/java/util/UUID.html}{\texttt{ Java}} and \href{http://docs.python.org/3/library/uuid.html}{\texttt{ Python}} UUID libraries.\hypertarget{md_vendor_2ramsey_2uuid_2_r_e_a_d_m_e_autotoc_md1627}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{md_vendor_2ramsey_2uuid_2_r_e_a_d_m_e_autotoc_md1627}
