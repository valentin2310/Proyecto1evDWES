It manages an environment made of application kernel and bundles.

{\footnotesize\ttfamily locate\+Resource (\begin{DoxyParamCaption}\item[{string}]{\$name }\end{DoxyParamCaption})}

Returns the file path for a given bundle resource.

A Resource can be a file or a directory.

The resource name must follow the following pattern\+: \begin{DoxyVerb}"@BundleName/path/to/a/file.something"
\end{DoxyVerb}
 where Bundle\+Name is the name of the bundle and the remaining part is the relative path in the bundle.
