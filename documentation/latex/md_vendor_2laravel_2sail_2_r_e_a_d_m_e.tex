Sail provides a Docker powered local development experience for Laravel that is compatible with mac\+OS, Windows (WSL2), and Linux. Other than Docker, no software or libraries are required to be installed on your local computer before using Sail. Sail\textquotesingle{}s simple CLI means you can start building your Laravel application without any previous Docker experience.\hypertarget{md_vendor_2laravel_2sail_2_r_e_a_d_m_e_autotoc_md923}{}\doxyparagraph{\texorpdfstring{Inspiration}{Inspiration}}\label{md_vendor_2laravel_2sail_2_r_e_a_d_m_e_autotoc_md923}
