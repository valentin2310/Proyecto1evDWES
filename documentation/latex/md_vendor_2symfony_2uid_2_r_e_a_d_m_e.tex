The UID component provides an object-\/oriented API to generate and represent UIDs.

It provides implementations that work on 32-\/bit and 64-\/bit CPUs for ULIDs and for UUIDs version 1 and versions 3 to 8.\hypertarget{md_vendor_2symfony_2uid_2_r_e_a_d_m_e_autotoc_md2375}{}\doxysection{\texorpdfstring{Resources}{Resources}}\label{md_vendor_2symfony_2uid_2_r_e_a_d_m_e_autotoc_md2375}
