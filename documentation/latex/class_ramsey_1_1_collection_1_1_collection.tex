\doxysubsection{Detailed Description}
A collection represents a group of objects.

Each object in the collection is of a specific, defined type.

This is a direct implementation of {\ttfamily Collection\+Interface}, provided for the sake of convenience.

Example usage\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\ php}

\DoxyCodeLine{\mbox{\hyperlink{namespace_symfony_1_1_component_1_1_routing_1_1_loader_1_1_configurator_1_1_traits_ab9e21dbbe588414048003c715034e2aa}{\$collection}}-\/>add(\textcolor{keyword}{new}\ \(\backslash\)My\(\backslash\)Foo());}
\DoxyCodeLine{\mbox{\hyperlink{namespace_symfony_1_1_component_1_1_routing_1_1_loader_1_1_configurator_1_1_traits_ab9e21dbbe588414048003c715034e2aa}{\$collection}}-\/>add(\textcolor{keyword}{new}\ \(\backslash\)My\(\backslash\)Foo());}
