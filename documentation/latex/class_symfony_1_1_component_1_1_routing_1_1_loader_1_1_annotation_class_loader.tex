\doxysubsection{Detailed Description}
Annotation\+Class\+Loader loads routing information from a PHP class and its methods.

You need to define an implementation for the configure\+Route() method. Most of the time, this method should define some PHP callable to be called for the route (a controller in MVC speak).

The @\+Route annotation can be set on the class (for global parameters), and on each method.

The @\+Route annotation main value is the route path. The annotation also recognizes several parameters\+: requirements, options, defaults, schemes, methods, host, and name. The name parameter is mandatory. Here is an example of how you should be able to use it\+: /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}
\begin{DoxyItemize}
\item @\+Route("{}/\+Blog"{})
\item / class Blog \{ /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}
\begin{DoxyItemize}
\item @\+Route("{}/"{}, name="{}blog\+\_\+index"{})
\item / public function index() \{ \} /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}
\item @\+Route("{}/\{id\}"{}, name="{}blog\+\_\+post"{}, requirements = \{"{}id"{} = "{}\textbackslash{}d+"{}\})
\item / public function show() \{ \} \}
\end{DoxyItemize}
\end{DoxyItemize}
