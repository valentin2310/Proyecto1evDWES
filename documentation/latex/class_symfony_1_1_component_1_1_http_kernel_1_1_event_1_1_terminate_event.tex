\doxysubsection{Detailed Description}
Allows to execute logic after a response was sent.

Since it\textquotesingle{}s only triggered on main requests, the {\ttfamily get\+Request\+Type()} method will always return the value of {\ttfamily Http\+Kernel\+Interface\+::\+MAIN\+\_\+\+REQUEST}.
