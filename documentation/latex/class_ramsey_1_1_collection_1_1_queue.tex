{\footnotesize\ttfamily offer (\begin{DoxyParamCaption}\item[{mixed}]{\$element }\end{DoxyParamCaption})}

Inserts the specified element into this queue if it is possible to do so immediately without violating capacity restrictions.

When using a capacity-\/restricted queue, this method is generally preferable to {\ttfamily add()}, which can fail to insert an element only by throwing an exception.
