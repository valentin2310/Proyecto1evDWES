{\bfseries{league/commonmark}} is a highly-\/extensible PHP Markdown parser created by \href{https://www.twitter.com/colinodell}{\texttt{ Colin O\textquotesingle{}Dell}} which supports the full \href{http://commonmark.org/}{\texttt{ Common\+Mark}} spec and \href{https://github.github.com/gfm/}{\texttt{ Git\+Hub-\/\+Flavored Markdown}}. It is based on the \href{https://github.com/jgm/commonmark.js}{\texttt{ Common\+Mark JS reference implementation}} by \href{http://johnmacfarlane.net}{\texttt{ John Mac\+Farlane}} (\href{https://github.com/jgm}{\texttt{ @jgm}}).\hypertarget{md_vendor_2league_2commonmark_2_r_e_a_d_m_e_autotoc_md1035}{}\doxysection{\texorpdfstring{üì¶ Installation \& Basic Usage}{üì¶ Installation \& Basic Usage}}\label{md_vendor_2league_2commonmark_2_r_e_a_d_m_e_autotoc_md1035}

Contributions to this library are {\bfseries{welcome}}, especially ones that\+:


\begin{DoxyItemize}
\item Improve usability or flexibility without compromising our ability to adhere to the \href{http://spec.commonmark.org/}{\texttt{ Common\+Mark spec}}
\item Mirror fixes made to the \href{https://github.com/jgm/commonmark.js}{\texttt{ reference implementation}}
\item Optimize performance
\item Fix issues with adhering to the \href{http://spec.commonmark.org/}{\texttt{ Common\+Mark spec}}
\end{DoxyItemize}

Major refactoring to core parsing logic should be avoided if possible so that we can easily follow updates made to \href{https://github.com/jgm/commonmark.js}{\texttt{ the reference implementation}}. That being said, we will absolutely consider changes which don\textquotesingle{}t deviate too far from the reference spec or which are favored by other popular Common\+Mark implementations.
