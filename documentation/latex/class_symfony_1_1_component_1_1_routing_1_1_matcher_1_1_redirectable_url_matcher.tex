{\footnotesize\ttfamily match (\begin{DoxyParamCaption}\item[{string}]{\$pathinfo }\end{DoxyParamCaption})}

Tries to match a URL path with a set of routes.

If the matcher cannot find information, it must throw one of the exceptions documented below.


\begin{DoxyParams}[1]{Parameters}
string & {\em \$pathinfo} & The path info to be parsed (raw format, i.\+e. not urldecoded)\\
\hline
\end{DoxyParams}

\begin{DoxyExceptions}{Exceptions}
{\em No\+Configuration\+Exception} & If no routing configuration could be found \\
\hline
{\em Resource\+Not\+Found\+Exception} & If the resource could not be found \\
\hline
{\em Method\+Not\+Allowed\+Exception} & If the resource was found but the request method is not allowed \\
\hline
\end{DoxyExceptions}
