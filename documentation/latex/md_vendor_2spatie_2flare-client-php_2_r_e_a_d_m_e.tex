This repository contains the PHP client to send errors and exceptions to \href{https://flareapp.io}{\texttt{ Flare}}. The client can be installed using composer and works for PHP 8.\+0 and above.

Using Laravel? You probably want to use \href{https://github.com/spatie/laravel-ignition}{\texttt{ Ignition for Laravel}}. It comes with a beautiful error page and has the Flare client built in.
