{\footnotesize\ttfamily capitalize (\begin{DoxyParamCaption}\item[{string}]{\$string,  }\item[{string}]{\$delimiters = {\ttfamily "{}~\textbackslash{}n\textbackslash{}t\textbackslash{}r\textbackslash{}0\textbackslash{}x0B-\/"{}} }\end{DoxyParamCaption})}

Uppercases words with configurable delimiters between words.

Takes a string and capitalizes all of the words, like PHP\textquotesingle{}s built-\/in ucwords function. This extends that behavior, however, by allowing the word delimiters to be configured, rather than only separating on whitespace.
