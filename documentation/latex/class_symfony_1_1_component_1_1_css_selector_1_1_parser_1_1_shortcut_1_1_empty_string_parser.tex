\doxysubsection{Detailed Description}
CSS selector class parser shortcut.

This shortcut ensure compatibility with previous version.
\begin{DoxyItemize}
\item The parser fails to parse an empty string.
\item In the previous version, an empty string matches each tags.
\end{DoxyItemize}
