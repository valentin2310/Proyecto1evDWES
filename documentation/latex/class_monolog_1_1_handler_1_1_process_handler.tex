\doxysubsection{Detailed Description}
Stores to STDIN of any process, specified by a command.

Usage example\+: 
\begin{DoxyPre}
\$log = new Logger('myLogger');
\$log-\/>pushHandler(new ProcessHandler('/usr/bin/php /var/www/monolog/someScript.php'));
\end{DoxyPre}
