\doxysubsection{Detailed Description}
Inflects based on multiple rulesets.

Rules\+:
\begin{DoxyItemize}
\item If the word matches any uninflected word pattern, it is not inflected
\item The first ruleset that returns a different value for an irregular word wins
\item The first ruleset that returns a different value for a regular word wins
\item If none of the above match, the word is left as-\/is 
\end{DoxyItemize}
