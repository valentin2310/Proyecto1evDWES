\doxylink{namespace_faker}{Faker} is a PHP library that generates fake data for you. Whether you need to bootstrap your database, create good-\/looking XML documents, fill-\/in your persistence to stress test it, or anonymize data taken from a production service, \doxylink{namespace_faker}{Faker} is for you.

It\textquotesingle{}s heavily inspired by Perl\textquotesingle{}s \href{https://metacpan.org/pod/Data::Faker}{\texttt{ Data\+::\+Faker}}, and by Ruby\textquotesingle{}s \href{https://rubygems.org/gems/faker}{\texttt{ Faker}}.\hypertarget{md_vendor_2fakerphp_2faker_2_r_e_a_d_m_e_autotoc_md218}{}\doxysubsection{\texorpdfstring{Getting Started}{Getting Started}}\label{md_vendor_2fakerphp_2faker_2_r_e_a_d_m_e_autotoc_md218}

\DoxyCodeLine{echo\ \$faker-\/>email();}
\DoxyCodeLine{\textcolor{comment}{//\ 'walter.sophia@hotmail.com'}}

\DoxyCodeLine{\textcolor{comment}{//\ 'Numquam\ ut\ mollitia\ at\ consequuntur\ inventore\ dolorem.'}}

If you already used this library with its properties, they are now deprecated and needs to be replaced by their equivalent methods.

You can use the provided \href{https://github.com/rectorphp/rector}{\texttt{ Rector}} config file to automate the work.

Run


\begin{DoxyCode}{0}
\DoxyCodeLine{composer\ require\ -\/-\/dev\ rector/rector}

\end{DoxyCode}


to install {\ttfamily rector/rector}.

Run


\begin{DoxyCode}{0}
\DoxyCodeLine{vendor/bin/rector\ process\ src/\ -\/-\/config\ vendor/fakerphp/faker/rector-\/migrate.php}

\end{DoxyCode}


to run {\ttfamily rector/rector}.

{\itshape Note\+:} do not forget to replace {\ttfamily src/} with the path to your source directory.

Alternatively, import the configuration in your {\ttfamily rector.\+php} file\+:


\begin{DoxyCode}{0}
