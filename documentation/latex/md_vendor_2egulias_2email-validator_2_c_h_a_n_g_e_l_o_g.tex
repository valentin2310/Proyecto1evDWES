\begin{DoxyItemize}
\item PHP version upgraded to match Symfony\textquotesingle{}s (as of 12/2020).
\item DNSCheck\+Validation now fails for missing MX records. While the RFC argues that the existence of only A records to be valid, starting in v3 they will be considered invalid.
\item Emails domain part are now intenteded to be RFC 1035 compliant, rendering previous valid emails (e.\+g example@examp\&) invalid.
