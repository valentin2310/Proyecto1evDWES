\chapter{What do all those files mean?}
\hypertarget{md_vendor_2nikic_2php-parser_2grammar_2_r_e_a_d_m_e}{}\label{md_vendor_2nikic_2php-parser_2grammar_2_r_e_a_d_m_e}\index{What do all those files mean?@{What do all those files mean?}}

\begin{DoxyItemize}
\item {\ttfamily php5.\+y}\+: PHP 5 grammar written in a pseudo language
\item {\ttfamily php7.\+y}\+: PHP 7 grammar written in a pseudo language
\item {\ttfamily tokens.\+y}\+: Tokens definition shared between PHP 5 and PHP 7 grammars
\item {\ttfamily parser.\+template}\+: A {\ttfamily kmyacc} parser prototype file for PHP
\item {\ttfamily tokens.\+template}\+: A {\ttfamily kmyacc} prototype file for the {\ttfamily Tokens} class
\item {\ttfamily \doxylink{rebuild_parsers_8php}{rebuild\+Parsers.\+php}}\+: Preprocesses the grammar and builds the parser using {\ttfamily kmyacc}
\end{DoxyItemize}\hypertarget{md_vendor_2nikic_2php-parser_2grammar_2_r_e_a_d_m_e_autotoc_md1312}{}\doxysection{\texorpdfstring{.\+phpy pseudo language}{.phpy pseudo language}}\label{md_vendor_2nikic_2php-parser_2grammar_2_r_e_a_d_m_e_autotoc_md1312}
The {\ttfamily .y} file is a normal grammar in {\ttfamily kmyacc} ({\ttfamily yacc}) style, with some transformations applied to it\+:


\begin{DoxyItemize}
\item Nodes are created using the syntax {\ttfamily Name\mbox{[}..., ...\mbox{]}}. This is transformed into {\ttfamily new Name(..., ..., attributes())}
\item Some function-\/like constructs are resolved (see {\ttfamily \doxylink{rebuild_parsers_8php}{rebuild\+Parsers.\+php}} for a list)
\end{DoxyItemize}\hypertarget{md_vendor_2nikic_2php-parser_2grammar_2_r_e_a_d_m_e_autotoc_md1313}{}\doxysection{\texorpdfstring{Building the parser}{Building the parser}}\label{md_vendor_2nikic_2php-parser_2grammar_2_r_e_a_d_m_e_autotoc_md1313}
Run {\ttfamily php \doxylink{rebuild_parsers_8php}{grammar/rebuild\+Parsers.\+php}} to rebuild the parsers. Additional options\+:


\begin{DoxyItemize}
\item The {\ttfamily KMYACC} environment variable can be used to specify an alternative {\ttfamily kmyacc} binary. By default the {\ttfamily phpyacc} dev dependency will be used. To use the original {\ttfamily kmyacc}, you need to compile \href{https://github.com/moriyoshi/kmyacc-forked}{\texttt{ moriyoshi\textquotesingle{}s fork}}.
\item The {\ttfamily -\/-\/debug} option enables emission of debug symbols and creates the {\ttfamily y.\+output} file.
\item The {\ttfamily -\/-\/keep-\/tmp-\/grammar} option preserves the preprocessed grammar file. 
\end{DoxyItemize}
