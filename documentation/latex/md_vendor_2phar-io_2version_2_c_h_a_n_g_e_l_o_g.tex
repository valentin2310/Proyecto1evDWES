\item \href{https://github.com/phar-io/version/issues/11}{\texttt{ \#11}}\+: Added support for pre-\/release version suffixes. Supported values are\+:
\begin{DoxyItemize}
\item {\ttfamily dev}
\item {\ttfamily beta} (also abbreviated form {\ttfamily b})
\item {\ttfamily rc}
\item {\ttfamily alpha} (also abbreviated form {\ttfamily a})
\item {\ttfamily patch} (also abbreviated form {\ttfamily p})
\end{DoxyItemize}

When comparing versions, the pre-\/release suffix is taken into account. Example\+: {\ttfamily 1.\+5.\+0 \texorpdfstring{$>$}{>} 1.\+5.\+0-\/beta1 \texorpdfstring{$>$}{>} 1.\+5.\+0-\/alpha3 \texorpdfstring{$>$}{>} 1.\+5.\+0-\/alpha2 \texorpdfstring{$>$}{>} 1.\+5.\+0-\/dev11}
