\doxysubsection{Detailed Description}
Evaluates constant expressions.

This evaluator is able to evaluate all constant expressions (as defined by PHP), which can be evaluated without further context. If a subexpression is not of this type, a user-\/provided fallback evaluator is invoked. To support all constant expressions that are also supported by PHP (and not already handled by this class), the fallback evaluator must be able to handle the following node types\+:


\begin{DoxyItemize}
\item All Scalar\textbackslash{}\+Magic\+Const\texorpdfstring{$\ast$}{*} nodes.
\item Expr\textbackslash{}\+Const\+Fetch nodes. Only null/false/true are already handled by this class.
\item Expr\textbackslash{}\+Class\+Const\+Fetch nodes.
\end{DoxyItemize}

The fallback evaluator should throw Const\+Expr\+Evaluation\+Exception for nodes it cannot evaluate.

The evaluation is dependent on runtime configuration in two respects\+: Firstly, floating point to string conversions are affected by the precision ini setting. Secondly, they are also affected by the LC\+\_\+\+NUMERIC locale.
