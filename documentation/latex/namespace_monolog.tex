Represents the log levels

\doxylink{namespace_monolog}{Monolog} supports the logging levels described by RFC 5424 \{\begin{DoxySeeAlso}{See also}
\href{https://datatracker.ietf.org/doc/html/rfc5424}{\texttt{ https\+://datatracker.\+ietf.\+org/doc/html/rfc5424}}\} but due to BC the severity values used internally are not 0-\/7.
\end{DoxySeeAlso}
To get the level name/value out of a Level there are several options\+:


\begin{DoxyItemize}
\item Use -\/\texorpdfstring{$>$}{>}get\+Name() to get the standard \doxylink{namespace_monolog}{Monolog} name which is full uppercased (e.\+g. "{}\+DEBUG"{})
\item Use -\/\texorpdfstring{$>$}{>}to\+Psr\+Log\+Level() to get the standard PSR-\/3 name which is full lowercased (e.\+g. "{}debug"{})
\item Use -\/\texorpdfstring{$>$}{>}to\+RFC5424\+Level() to get the standard RFC 5424 value (e.\+g. 7 for debug, 0 for emergency)
\item Use -\/\texorpdfstring{$>$}{>}name to get the enum case\textquotesingle{}s name which is capitalized (e.\+g. "{}\+Debug"{})
\end{DoxyItemize}
