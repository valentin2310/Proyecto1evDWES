Lets you reload the static array from another file

This is only useful for complex integrations in which a project needs to use this class but then also needs to execute another project\textquotesingle{}s autoloader in process, and wants to ensure both projects have access to their version of \doxylink{installed_8php}{installed.\+php}.

A typical case would be \doxylink{namespace_p_h_p_unit}{PHPUnit}, where it would need to make sure it reads all the data it needs from this class, then call reload() with {\ttfamily require \$CWD/vendor/composer/installed.\+php} (or similar) as input to make sure the project in which it runs can then also use this class safely, without interference between \doxylink{namespace_p_h_p_unit}{PHPUnit}\textquotesingle{}s dependencies and the project\textquotesingle{}s dependencies.
