\doxysubsection{Detailed Description}
Returns an asynchronous response using curl\+\_\+multi\+\_\+\texorpdfstring{$\ast$}{*} functions.

When using the Curl\+Multi\+Handler, custom curl options can be specified as an associative array of curl option constants mapping to values in the {\bfseries{curl}} key of the provided request options.
