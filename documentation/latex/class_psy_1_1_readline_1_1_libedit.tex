\doxysubsection{Detailed Description}
A Libedit-\/based Readline implementation.

This is largely the same as the Readline implementation, but it emulates support for {\ttfamily readline\+\_\+list\+\_\+history} since PHP decided it was a good idea to ship a fake Readline implementation that is missing history support.

NOTE\+: As of PHP 7.\+4, PHP sometimes has history support in the Libedit wrapper, so it will use the GNUReadline implementation rather than this one.
