\doxysubsection{Detailed Description}
A simple class used internally by Psy\+SH to represent silent input.

Silent input is generally used for non-\/user-\/generated code, such as the rewritten user code run by sudo command. Silent input isn\textquotesingle{}t echoed before evaluating, and it\textquotesingle{}s not added to the readline history.
