{\footnotesize\ttfamily static resolve (\begin{DoxyParamCaption}\item[{}]{\$file,  }\item[{}]{\$line\+Number = {\ttfamily null} }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Runtime validation that \$file can be resolved as an include path.

If \$file can be resolved, return \$file. Otherwise throw a fatal error exception.

If \$file collides with a path in the currently running Psy\+SH phar, it will be resolved relative to the include path, to prevent PHP from grabbing the phar version of the file.
