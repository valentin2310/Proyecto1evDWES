{\ttfamily Doctrine\textbackslash{}Common\textbackslash{}Lexer\textbackslash{}Token} no longer implements {\ttfamily Array\+Access}. Parameter type declarations have been added to {\ttfamily Doctrine\textbackslash{}Common\textbackslash{}Lexer\textbackslash{}Abstract\+Lexer} and {\ttfamily Doctrine\textbackslash{}Common\textbackslash{}Lexer\textbackslash{}Token}. You should add both parameter type declarations and return type declarations to your lexers, based on the {\ttfamily @return} phpdoc.\hypertarget{md_vendor_2doctrine_2lexer_2_u_p_g_r_a_d_e_autotoc_md88}{}\doxysection{\texorpdfstring{Upgrade to 2.\+0.\+0}{Upgrade to 2.0.0}}\label{md_vendor_2doctrine_2lexer_2_u_p_g_r_a_d_e_autotoc_md88}
{\ttfamily Abstract\+Lexer\+::glimpse()} and {\ttfamily Abstract\+Lexer\+::peek()} now return instances of {\ttfamily Doctrine\textbackslash{}Common\textbackslash{}Lexer\textbackslash{}Token}, which is an array-\/like class Using it as an array is deprecated in favor of using properties of that class. Using {\ttfamily count()} on it is deprecated with no replacement.
