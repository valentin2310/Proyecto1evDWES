\doxysubsection{Detailed Description}
This extension sub-\/class provides first-\/class integration with the Config/\+Definition Component.

You can use this as base class if

a) you use the Config/\+Definition component for configuration, b) your configuration class is named "{}\+Configuration"{}, and c) the configuration class resides in the Dependency\+Injection sub-\/folder.
