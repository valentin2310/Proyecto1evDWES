\doxysubsection{Detailed Description}
A Input\+Definition represents a set of valid command line arguments and options.

Usage\+: \begin{DoxyVerb}$definition = new InputDefinition([
    new InputArgument('name', InputArgument::REQUIRED),
    new InputOption('foo', 'f', InputOption::VALUE_REQUIRED),
]);
\end{DoxyVerb}
