Clients accept an array of constructor parameters.

Here\textquotesingle{}s an example of creating a client using a base\+\_\+uri and an array of default request options to apply to each request\+: \begin{DoxyVerb}$client = new Client([
    'base_uri'        => 'http://www.foo.com/1.0/',
    'timeout'         => 0,
    'allow_redirects' => false,
    'proxy'           => '192.168.16.1:10'
]);
\end{DoxyVerb}
 Client configuration settings include the following options\+:


\begin{DoxyItemize}
\item handler\+: (callable) Function that transfers HTTP requests over the wire. The function is called with a Psr7\textbackslash{}\+Http\textbackslash{}\+Message\textbackslash{}\+Request\+Interface and array of transfer options, and must return a \doxylink{namespace_guzzle_http}{Guzzle\+Http}\textbackslash{}\+Promise\textbackslash{}\+Promise\+Interface that is fulfilled with a Psr7\textbackslash{}\+Http\textbackslash{}\+Message\textbackslash{}\+Response\+Interface on success. If no handler is provided, a default handler will be created that enables all of the request options below by attaching all of the default middleware to the handler.
\item base\+\_\+uri\+: (string\texorpdfstring{$\vert$}{|}\+Uri\+Interface) Base URI of the client that is merged into relative URIs. Can be a string or instance of Uri\+Interface.
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+: any request option
\end{DoxyItemize}
