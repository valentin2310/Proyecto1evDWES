{\footnotesize\ttfamily get\+Client\+Ip (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Returns the client IP address.

This method can read the client IP address from the "{}\+X-\/\+Forwarded-\/\+For"{} header when trusted proxies were set via "{}set\+Trusted\+Proxies()"{}. The "{}\+X-\/\+Forwarded-\/\+For"{} header value is a comma+space separated list of IP addresses, the left-\/most being the original client, and each successive proxy that passed the request adding the IP address where it received the request from.

If your reverse proxy uses a different header name than "{}\+X-\/\+Forwarded-\/\+For"{}, ("{}\+Client-\/\+Ip"{} for instance), configure it via the \$trusted\+Header\+Set argument of the Request\+::set\+Trusted\+Proxies() method instead.
