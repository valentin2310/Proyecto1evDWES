{\footnotesize\ttfamily fix\+Php\+Files\+Array (\begin{DoxyParamCaption}\item[{array}]{\$data }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [protected]}}

Fixes a malformed PHP \$\+\_\+\+FILES array.

PHP has a bug that the format of the \$\+\_\+\+FILES array differs, depending on whether the uploaded file fields had normal field names or array-\/like field names ("{}normal"{} vs. "{}parent\mbox{[}child\mbox{]}"{}).

This method fixes the array to look like the "{}normal"{} \$\+\_\+\+FILES array.

It\textquotesingle{}s safe to pass an already converted array, in which case this method just returns the original array unmodified.
