\doxysubsection{Detailed Description}
Base class for output classes.

There are five levels of verbosity\+:


\begin{DoxyItemize}
\item normal\+: no option passed (normal output)
\item verbose\+: -\/v (more output)
\item very verbose\+: -\/vv (highly extended output)
\item debug\+: -\/vvv (all debug output)
\item quiet\+: -\/q (no output)
\end{DoxyItemize}
