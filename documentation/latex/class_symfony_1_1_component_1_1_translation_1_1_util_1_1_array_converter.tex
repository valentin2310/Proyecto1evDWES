\doxysubsection{Detailed Description}
Array\+Converter generates tree like structure from a message catalogue. e.\+g. this \textquotesingle{}foo.\+bar1\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}test1\textquotesingle{}, \textquotesingle{}foo.\+bar2\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}test2\textquotesingle{} converts to follows\+: foo\+: bar1\+: test1 bar2\+: test2.
