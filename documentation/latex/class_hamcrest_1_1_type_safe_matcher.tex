\doxysubsection{Detailed Description}
Convenient base class for Matchers that require a value of a specific type. This simply checks the type.

While it may seem a useless exercise to have this in PHP, objects cannot be cast to certain data types such as numerics (or even strings if \+\_\+\+\_\+to\+String() has not be defined).
