\doxysubsection{Detailed Description}
Console\+Output is the default class for all CLI output. It uses STDOUT and STDERR.

This class is a convenient wrapper around {\ttfamily Stream\+Output} for both STDOUT and STDERR. \begin{DoxyVerb}$output = new ConsoleOutput();
\end{DoxyVerb}
 This is equivalent to\+: \begin{DoxyVerb}$output = new StreamOutput(fopen('php://stdout', 'w'));
$stdErr = new StreamOutput(fopen('php://stderr', 'w'));
\end{DoxyVerb}
