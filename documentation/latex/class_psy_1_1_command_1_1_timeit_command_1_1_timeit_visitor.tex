\doxysubsection{Detailed Description}
A node visitor for instrumenting code to be executed by the {\ttfamily timeit} command.

Injects {\ttfamily Timeit\+Command\+::mark\+Start()} at the start of code to be executed, and {\ttfamily Timeit\+Command\+::mark\+End()} at the end, and on top-\/level return statements.
