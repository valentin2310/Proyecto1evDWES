Note that if the current Request has no surrogate capability, this method falls back to use the inline rendering strategy.

Additional available options\+:


\begin{DoxyItemize}
\item alt\+: an alternative URI to render in case of an error
\item comment\+: a comment to add when returning the surrogate tag
\item absolute\+\_\+uri\+: whether to generate an absolute URI or not. Default is false
\end{DoxyItemize}

Note, that not all surrogate strategies support all options. For now \textquotesingle{}alt\textquotesingle{} and \textquotesingle{}comment\textquotesingle{} are only supported by ESI.
