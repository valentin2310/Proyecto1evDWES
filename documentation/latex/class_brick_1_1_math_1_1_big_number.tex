Creates a Big\+Number of the given value.

The concrete return type is dependent on the given value, with the following rules\+:


\begin{DoxyItemize}
\item Big\+Number instances are returned as is
\item integer numbers are returned as Big\+Integer
\item floating point numbers are converted to a string then parsed as such
\item strings containing a {\ttfamily /} character are returned as Big\+Rational
\item strings containing a {\ttfamily .} character or using an exponential notation are returned as Big\+Decimal
\item strings containing only digits with an optional leading {\ttfamily +} or {\ttfamily -\/} sign are returned as Big\+Integer
\end{DoxyItemize}

Note that this method can discard information as the precision of a floating-\/point value is inherently limited.
