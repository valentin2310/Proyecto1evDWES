\href{https://github.com/guzzle/guzzle/releases}{\texttt{ }} \href{https://github.com/guzzle/guzzle/actions?query=workflow\%3ACI}{\texttt{ }} \href{https://packagist.org/packages/guzzlehttp/guzzle}{\texttt{ }}

Guzzle is a PHP HTTP client that makes it easy to send HTTP requests and trivial to integrate with web services.


\begin{DoxyItemize}
\item Simple interface for building query strings, POST requests, streaming large uploads, streaming large downloads, using HTTP cookies, uploading JSON data, etc...
\item Can send both synchronous and asynchronous requests using the same interface.
\item Uses PSR-\/7 interfaces for requests, responses, and streams. This allows you to utilize other PSR-\/7 compatible libraries with Guzzle.
\item Supports PSR-\/18 allowing interoperability between other PSR-\/18 HTTP Clients.
\item Abstracts away the underlying HTTP transport, allowing you to write environment and transport agnostic code; i.\+e., no hard dependency on c\+URL, PHP streams, sockets, or non-\/blocking event loops.
\item Middleware system allows you to augment and compose client behavior.
\end{DoxyItemize}
